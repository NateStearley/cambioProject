\doxysubsection{Detailed Description}
A decorator class that wraps a \mbox{\hyperlink{classDrone}{Drone}} implementation, forwarding all method calls to the host drone except \mbox{\hyperlink{classIEntity_a55992d3fff30c3e47fb53d44097d9a19}{Drone\+::\+Set\+Graph}} and \mbox{\hyperlink{classDrone_a11e525c730ee22bbd966127b9a99b3c9}{Drone\+::\+Update}}. The purpose of this class is to indiscriminately override these two functions regardless of the implementation of a given \mbox{\hyperlink{classDrone}{Drone}}.
